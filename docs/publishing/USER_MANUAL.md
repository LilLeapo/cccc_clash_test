# Mihomo Flutter Cross 用户手册

## 欢迎使用Mihomo Flutter Cross

Mihomo Flutter Cross是一款现代化的跨平台代理客户端，支持Android、iOS、Windows、macOS和Linux平台。基于先进的Flutter技术栈，提供流畅的用户体验和强大的代理功能。

## 主要功能

### 🚀 跨平台TUN代理
- **Android**: 基于VpnService的高效TUN模式
- **iOS**: 使用NEPacketTunnelProvider的原生VPN体验
- **桌面端**: Windows/macOS/Linux完整支持

### 🎨 现代化界面
- **Material 3设计**: 符合最新设计规范的精美界面
- **实时流量监控**: 可视化展示网络流量和连接状态
- **智能仪表板**: 直观的数据展示和快速操作

### ⚙️ 灵活配置
- **YAML配置支持**: 完整的Clash配置文件兼容
- **实时配置编辑**: 即时预览和验证配置变更
- **配置导入导出**: 便捷的配置文件管理

### 💾 安全存储
- **本地数据库**: Hive数据库加密存储
- **配置同步**: 跨设备配置同步
- **隐私保护**: 最小化数据收集

## 安装指南

### Android安装

#### Google Play Store (推荐)
1. 搜索"Mihomo Flutter Cross"
2. 点击"安装"
3. 安装完成后打开应用

#### 直接下载APK
1. 从官方网站下载最新APK文件
2. 允许"未知来源"安装
3. 点击APK文件进行安装
4. 安装完成后打开应用

### iOS安装

#### App Store
1. 打开App Store
2. 搜索"Mihomo Flutter Cross"
3. 点击"获取"进行安装
4. 安装完成后打开应用

#### TestFlight (测试版)
1. 下载TestFlight应用
2. 使用邀请链接加入测试
3. 下载测试版本

### 桌面端安装

#### Windows
1. 下载Windows版本安装程序
2. 运行安装程序并按照提示安装
3. 启动程序

#### macOS
1. 下载macOS版本应用包
2. 将应用拖拽到Applications文件夹
3. 启动程序

#### Linux
1. 下载AppImage或对应软件包
2. 设置执行权限: `chmod +x mihomo-flutter-cross.AppImage`
3. 运行应用

## 快速开始

### 1. 首次启动
1. 打开应用
2. 阅读并同意服务条款
3. 允许必要的权限请求

### 2. 配置代理

#### 使用配置文件
1. 点击"配置"标签
2. 选择"导入配置"
3. 上传YAML配置文件
4. 配置自动加载完成

#### 手动配置
1. 点击"添加服务器"
2. 填入服务器信息:
   - 服务器地址
   - 端口
   - 加密方式
   - 密码
3. 保存配置

### 3. 启动代理
1. 在主界面点击"连接"按钮
2. 确认VPN权限请求 (Android/iOS)
3. 等待连接成功提示
4. 查看实时流量统计

### 4. 流量监控
- **实时图表**: 查看上下行流量趋势
- **连接状态**: 监控服务器连接状态
- **流量统计**: 查看总流量和连接时长

#### 规则配置

### 设置选项

#### 代理设置
- **TUN模式**: 启用/禁用TUN模式
- **DNS设置**: 自定义DNS服务器

#### 界面设置
- **主题切换**: 明亮/暗黑主题
- **语言设置**: 多语言支持
- **通知设置**: 推送通知配置

#### 高级设置
- **日志级别**: 调试日志详细程度
- **网络接口**: 指定网络接口
- **内存优化**: 内存使用优化选项

## 使用技巧

### 性能优化
1. **选择最佳服务器**: 使用延迟测试选择低延迟服务器
2. **合理配置规则**: 避免过于复杂的规则影响性能
3. **定期重启**: 长时间运行后重启应用清理内存

### 网络优化
1. **DNS设置**: 使用快速DNS服务器
2. **连接重试**: 配置合理的重试间隔
3. **负载均衡**: 多服务器时启用负载均衡

### 故障排除
1. **连接失败**: 检查服务器信息和网络连接
2. **速度慢**: 更换服务器或检查网络带宽
3. **应用崩溃**: 清除缓存重启应用

## 常见问题

### 安装问题

**Q: Android安装时提示"未知来源"错误**
A: 进入设置 > 安全 > 允许安装未知来源应用

**Q: iOS无法安装**
A: 确保设备版本iOS 12.0+，检查存储空间是否足够

**Q: 桌面版无法启动**
A: 检查系统版本兼容性，重新下载对应版本

### 连接问题

**Q: 无法建立VPN连接**
A:
- 检查网络连接
- 验证服务器配置
- 尝试重启应用

**Q: 连接后无法上网**
A:
- 检查路由规则配置
- 确认DNS设置正确
- 尝试更换服务器

**Q: 连接速度很慢**
A:
- 测试不同服务器延迟
- 检查本地网络带宽
- 优化配置规则

### 功能问题

**Q: 配置导入失败**
A:
- 检查YAML文件格式
- 验证文件编码为UTF-8
- 确认配置内容完整

**Q: 实时统计不更新**
A:
- 重启应用
- 检查应用权限
- 清理后台进程

**Q: 应用内存占用过高**
A:
- 重启应用
- 减少同时连接的服务器
- 调整日志级别
